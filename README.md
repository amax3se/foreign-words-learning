# Foreign Words Learning
It is dictionary, that will help you to learn foreign words by using cards

## Instalation
1) Install the [library](https://github.com/nlohmann/json) for saving your progress in the file:
    - vcpkg (Windows, Linux, macOS): vcpkg install nlohmann-json.
    - Conan: conan install nlohmann_json/3.11.3@.
    - Homebrew (macOS): brew install nlohmann-json.
    - APT (Ubuntu/Debian): sudo apt install nlohmann-json3-dev
2) Clone repository
``` shell
git clone https://github.com/amax3se/foreign-words-learning
```
3) Compile main file
``` shell
g++ main.cpp -o main
```
4) Run main file
``` shell
./main
```
